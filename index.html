Country Top Trumps

Guess which country comes out on top in a key statistic!

How to Play

  • A random stat category appears between two countries.
  • Simply click the name of the country you think wins the category.
  • See how many you can get right in a row!

Project Information

  • Country Top Trumps is a front-end web game built with HTML, vanilla JavaScript and CSS.
  • The website contacts the REST Countries API to access information about random countries.
  • The game features: dynamic round generation, game state management, API error handling and a modern glass card style using a CSS grid layout.
